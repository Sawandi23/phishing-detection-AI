# Phishing Detection Using Artificial Intelligence

## Overview
This repository contains a mini research project focused on detecting phishing attempts using Artificial Intelligence techniques. The study explores how machine learning models can be applied to identify malicious patterns in phishing data and improve cybersecurity defenses.

This project was completed as part of an academic research component at **Sri Lanka Institute of Information Technology (SLIIT)**.

## Abstract
Phishing is one of the most common and damaging cyber threats, targeting users through deceptive emails, websites, and messages. Traditional rule-based detection methods often fail to adapt to evolving attack patterns.

This research investigates the use of AI and machine learning techniques to detect phishing more accurately by analyzing patterns in data such as URLs, email content, and behavioral features. The study highlights the effectiveness of intelligent models in improving detection accuracy while reducing false positives.

---

## Objectives
- Understand phishing attack techniques and their impact  
- Explore AI/ML-based approaches for phishing detection  
- Analyze features used to identify phishing attempts  
- Evaluate the potential of AI models in cybersecurity applications
